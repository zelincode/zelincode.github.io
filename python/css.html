# CSS选择器

  • 什么前缀都没有是标签 css.("div")
  • 前缀是“.”的是class css.(".right")
  • 前缀是"#"代表id

  • A,B代表A或B

  • A B 代表后代选择器（所有子代）

  • A>B 代表所有A中的B 子代选择器（直接子代）

    ​

  • A+B 所有和A紧邻的B
